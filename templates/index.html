Abishek's Project for Finacial Services Innovation Lab

This project was undertaken as an programming assignment for the financial services innovation lab. I utilized Gemini 1.5 PRO as the LLM for my analysis. The companies under scrutiny were Google and Nvidia, with a primary focus on researching their financials and future outlook.

The tech stack employed for this project was diverse and robust. I used Beautiful Soup 4 for web scraping information from the 10-K HTML files, which were downloaded using the SEC Downloader library. Data cleaning was performed using Pandas, and Matplotlib was used for graph plotting.

Flask was the framework of choice for creating a web application. I extracted the MD&A section from the 10-K reports and conducted an analysis using Gemini. This also extended to the creation of financial graphs, comparative analysis, and more.

The web application, designed to display the graphs and analysis, was built using Flask. The styling of the application was achieved using HTML and CSS. This project, therefore, served as a comprehensive exercise in data analysis, web development, and financial research.
